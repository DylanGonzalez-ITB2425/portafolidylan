Creación de esta Página Web

Este proyecto consiste en la creación de una página web utilizando HTML, CSS y JavaScript (JSS). El objetivo fue desarrollar una página funcional y visualmente atractiva, optimizada para diferentes tamaños de pantalla y dispositivos.

A continuación se explica el proceso de desarrollo, las tecnologías utilizadas y cómo se implementaron las funcionalidades.

Objetivos

Crear una página web interactiva con un diseño limpio y responsivo, asegurando que el contenido sea accesible y fácil de navegar en diferentes dispositivos.

Tecnologías Utilizadas

  • HTML: Estructura del contenido.
  • CSS: Estilo y diseño responsivo.
  • JavaScript (JSS): Funcionalidad dinámica para mejorar la interacción.

Lecciones Aprendidas

Aprendí a implementar una estructura organizada, mejorar la accesibilidad de la página y crear interactividad con JavaScript, además de practicar cómo trabajar con un diseño responsivo.
